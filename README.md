## Run Locally

Go to the project directory and install dependencies

```bash
  cd monoplay-v2.2
  npm install
```

Start the server

```bash
  npm run dev
```

## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

### Firebase

`VITE_APIKEY`
`VITE_AUTHDOMAIN`
`VITE_DATABASEURL`
`VITE_PROJECTID`
`VITE_STORAGEBUCKET`
`VITE_MESSAGINGSENDERID`
`VITE_APPID`
